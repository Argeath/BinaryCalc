Binary operations

In mathematics, a binary operation on a set is a calculation that combines two elements of the set (called operands) to produce another element of the set. In this calculator you can perform AND, OR and XOR operations.

AND operation means that only when all(2) input variables must be 1, only then result is 1.
x y x & y
0 0 0
1 0 0
0 1 0
1 1 1

OR operation means that whether any input variable is 1 then result is 1.
x y x | y
0 0 0
1 0 1
0 1 1
1 1 1
