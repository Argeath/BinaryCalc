Binary addition is much like your normal everyday addition (decimal addition), except that it carries on a value of 2 instead of a value of 10.
For example: in decimal addition, if you add 8 + 4 you get "12", which is 2 and carry = 1.
Same thing happens in binary addition, if you add 1 + 1 you get "10" - which is 0 and carry = 1.
